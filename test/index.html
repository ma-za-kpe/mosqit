üõ†Ô∏è Installation & Setup

  1. Enable Chrome AI: Navigate to chrome://flags and enable:
    • Prompt API for Gemini Nano
    • Summarization API for WebUI
    • Writer API for WebUI
    • Rewriter API for WebUI
    • Optimization Guide On Device Model
  2. Restart Chrome: Completely restart Chrome (quit from system tray)
  3. Clone Repository: git clone https://github.com/ma-za-kpe/mosqit.git
  4. Build Extension: npm install && npm run build
  5. Load Extension:
    • Open chrome://extensions
    • Enable Developer mode
    • Click "Load unpacked"
    • Select the dist folder
  6. Download AI Model: On first use, Chrome will download Gemini Nano (~2GB, takes 20-30 minutes)
